# 浪白漫画字幕工坊：AI 精准字幕脚本生成指南

这份规范用于让任何文本 AI 生成可直接导入“浪白漫画字幕工坊”的 `BCS顺序字幕脚本 v2`。软件只按图片出现顺序匹配，不按文件名匹配。

## 推荐操作

1. 在软件中导入图片并确认顺序。
2. 打开“字幕”窗口，点击“导出当前模板”。模板已包含每张图片的顺序、原文件名提示、准确原图尺寸和现有气泡 ID。
3. 把模板、图片内容说明和下方提示词一起发给 AI。
4. 要求 AI 只返回纯文本脚本，不要 Markdown 代码块或解释。
5. 将返回内容保存为 UTF-8 `.txt`，在软件中“导入 TXT”，再点击“应用并自动排版”。

## 可直接复制给 AI 的提示词

你是漫画字幕排版师。请根据我提供的图片顺序、原图尺寸、画面内容和对白，生成“BCS顺序字幕脚本 v2”。只输出完整纯文本脚本，不要代码块、解释、前后缀或遗漏任何图片段。

硬性规则：

- 第一行必须是 `@格式=BCS顺序字幕脚本`，第二行必须是 `@版本=2`，第三行必须是 `@坐标单位=px`。
- 每张图片必须恰好有一个 `[图片 N]` 段，N 从 1 连续递增，严格按输入顺序排列；不得按文件名重新排序。
- 每段必须原样保留 `@原图尺寸=宽x高`。所有矩形均使用该原图的像素坐标，左上角为 `(0,0)`。
- 每个气泡必须有唯一且稳定的 `@气泡ID`；已有 ID 必须原样保留，新气泡使用 `pN-bN`。
- 每个气泡依次输出：`@气泡ID`、`@矩形=x,y,宽,高`、`@尾巴`、`@气泡`、`@字体`、`@字号`、`@颜色`、`@行距`、`@描边`、`@白底透明度`，下一行写正文。
- `@气泡` 只能使用：`对话气泡`、`心理气泡`、`旁白框`、`耳语气泡`、`惊喊气泡`。
- `@尾巴` 只能使用：`左上`、`右上`、`左下`、`右下`。旁白框没有可见尾巴，但仍填写任一合法值。
- 气泡矩形必须完全位于图片内：`x>=0`、`y>=0`、`x+宽<=原图宽`、`y+高<=原图高`。
- 优先放在留白区，不得遮挡脸、眼睛、手、关键动作、重要物体和已有文字；气泡之间尽量不重叠。
- 尾巴方向应从气泡朝向说话者或心理活动者，软件会将尾部固定在对应角，不生成尾尖坐标。
- `@矩形` 是最终位置指令。点击“应用并自动排版”后，脚本矩形必须覆盖软件中该气泡此前的手工位置。
- 对话气泡用于普通对白；心理气泡使用云朵主体和由大到小的圆点尾；旁白框用于叙述；耳语气泡用于轻声或低语；惊喊气泡用于大喊、震惊或强烈情绪。
- 颜色为 6 位十六进制字体颜色，如 `#141518`。颜色只改变内部文字，不改变白色/浅灰底和黑色描边。
- `@白底透明度` 为 0–100 的整数；100 完全不透明，0 完全透明，仅影响气泡白色或浅灰底，不影响文字和描边。
- 默认字体 `Microsoft YaHei`、字号 `34`、行距 `1.25`、描边 `2`。文字较多时先增大气泡，再适度缩小字号，不得让文字溢出。
- 每个气泡块之间空一行。不得杜撰对白，不得修改原图内容。

## 完整示例

```text
@格式=BCS顺序字幕脚本
@版本=2
@坐标单位=px

[图片 1]
@原文件名=001.png
@原图尺寸=1080x1920

@气泡ID=p1-b1
@矩形=70,90,500,250
@尾巴=右下
@气泡=对话气泡
@字体=Microsoft YaHei
@字号=34
@颜色=#141518
@行距=1.25
@描边=2
你终于来了。

@气泡ID=p1-b2
@矩形=620,140,360,220
@尾巴=左下
@气泡=心理气泡
@字体=Microsoft YaHei
@字号=30
@颜色=#356DB5
@行距=1.25
@描边=2
他怎么会在这里？

[图片 2]
@原文件名=002.png
@原图尺寸=1080x1920

@气泡ID=p2-b1
@矩形=70,90,470,210
@尾巴=右下
@气泡=旁白框
@字体=Microsoft YaHei
@字号=32
@颜色=#141518
@行距=1.25
@描边=2
十分钟前。
```

## 导入前自检

- `[图片 N]` 数量与软件图片数量完全一致。
- 所有 `@原图尺寸` 与软件模板一致。
- 图片段顺序连续，没有重复、跳号或按文件名改序。
- 所有气泡 ID 唯一，矩形均在原图边界内。
- 气泡类型、尾巴方向、颜色均使用合法写法。
- 输出是 UTF-8 纯文本，没有 Markdown 围栏和 AI 解释。
