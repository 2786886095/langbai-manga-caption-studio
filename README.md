# 浪白漫画字幕工坊 0.9.0

全本地运行的批量漫画气泡字幕编辑器。Windows 同时提供 Setup 安装版与便携版；同一套 Flutter 核心可构建 Android、iOS、macOS、Linux 与 Web。

## 0.9.0 核心能力

- 项目页显示第一张图片缩略图，软件统一更名为“浪白漫画字幕工坊”。
- Setup 安装版支持启动检查、后台下载和软件内重启安装更新；便携版及其他平台会打开 GitHub Releases。
- 有新版时，项目页顶部显示醒目的整行更新提示和下载进度。
- 批量导出直接向所选目录写入 PNG，不再创建 ZIP；未设目录时主动询问，同名图覆盖前逐张确认。
- “排版”现在会明确重算已有气泡位置、保留样式并报告处理结果；无字幕时给出操作提示。
- 取消页面切换和流程卡片的装饰动画，批量渲染时逐张让出 UI 线程，降低卡顿。

- 软件内的 AI 生成指南直接读取 `AI字幕脚本生成指南.md`，并自动附加当前项目的精确图片清单与完整脚本模板。

- 交互性能修复：气泡未变化时不再重绘整张画布；自动保存只写轻量编辑层，不再重复编码全部原图。
- 编辑预览最长边降为 768px，原图尺寸、坐标和最终导出清晰度完全不变。
- 四角指向尾巴长度减半并收窄尾根，只保留简短清晰的指向。

- 大图编辑采用最长边 1600px 的轻量预览，导出时才逐张解码原图，显著降低批量加载后的内存与卡顿。
- 五种漫画样式：对话气泡、规范云朵心理气泡、旁白框、虚线耳语气泡、尖锐惊喊气泡。
- 全色域字体颜色选择器，支持 HSV 和 HEX 精确输入。
- 可导入 TTF/OTF 字体；字体数据随工程保存，重新打开后自动加载。
- 字幕窗口提供“AI 生成指南”，会自动写入当前图片顺序与准确尺寸并一键复制。
- 完整独立规范见 `AI字幕脚本生成指南.md`。

- 启动先进入本地项目页，可新建、删除和随时切换项目；名称留空时按创建时间命名。
- 项目独立保存图片、字幕和排版，工作中自动保存，切换项目时强制保存。
- 多选图片后默认按文件名自然排序，再进入“确认图片顺序”；支持拖拽重排，确认顺序是脚本、工程与导出的唯一顺序。
- 文件名只决定导入确认页的默认自然排序，不参与字幕匹配；同名图片可以共存。
- BCS 顺序字幕脚本 v2 使用 `[图片 N]` 对应第 N 张图片，并强制记录 `@原图尺寸=宽x高`。
- 精准气泡使用原图像素 `@矩形=x,y,宽,高`，应用前严格检查页数、图片尺寸和越界。
- 稳定 `@气泡ID` 确保在中间插入字幕后，已有气泡仍保留各自手调位置与样式。
- 对话气泡、旁白框、心理气泡；四角固定尾巴，心理气泡仅将三角尾巴替换为连续圆点。
- 字体、实色字体颜色、字号、行距、描边、拖动和八向缩放。
- 字幕编辑使用草稿事务，点击取消不会污染工程。
- `.bcs.json` v2 工程保存页面身份、确认顺序、原图尺寸和气泡 ID；v1 工程按原页面数组顺序迁移。
- 主流程精简为“图片 → 字幕 → 排版”；导出独立放在右上角。
- 设置可选择默认保存目录、是否每次询问、导出命名方式、自动保存开关与间隔。
- PNG 成图可按 `0001、0002……` 编号直接写入目录；若同名图片存在，可覆盖此图、全部覆盖、跳过或取消。
- 所有图片和字幕只在当前设备处理，不上传。

## 使用流程

1. 在项目页创建或选择项目；项目名留空会自动使用当前时间。
2. 点击“添加图片”。软件默认按文件名自然排序；可在顺序确认窗口拖动图片，然后点击“确认此顺序”。
3. 点击“字幕”，导出当前模板或导入 UTF-8 TXT。
4. 每个 `[图片 N]` 段对应确认后的第 N 张图片，并包含实际 `@原图尺寸`；文件名不参与匹配。
5. 点击“应用并自动排版”。页数、尺寸或矩形越界时会阻止应用。
6. 在画布上调整气泡，点击空白处可取消选框。
7. 使用右上角“批量导出”。若没有默认目录，软件会先询问导出文件夹，然后逐张写入 PNG；再次导出同名图片时会询问是否覆盖。

常用快捷键：`Ctrl+S` 保存、`Ctrl+O` 打开、`Ctrl+Z` 撤销、`Ctrl+Y` 重做、`Ctrl+D` 复制气泡、`Delete` 删除气泡。

## 字幕脚本示例

```text
@格式=BCS顺序字幕脚本
@版本=2
@坐标单位=px

[图片 1]
@原文件名=scene-B.png
@原图尺寸=1080x1920

@气泡ID=p1-b1
@矩形=80,100,520,260
@气泡=对话气泡
@尾巴=右下
@字体=Microsoft YaHei
@字号=34
@颜色=#141518
@行距=1.25
@描边=3
你迟到了整整十分钟！
```

完整规范见 [字幕导入格式规范.md](字幕导入格式规范.md)。旧版 `[文件名]` 脚本仍可导入，但只按段落出现顺序迁移，绝不会按文件名寻找图片。

## 本地开发与构建

```powershell
flutter pub get
flutter test
flutter run
```

Windows EXE 使用 Flutter Web Release 加 Electron 壳：

```powershell
flutter build web --release
(Get-Content build/web/index.html -Raw).Replace('<base href="/">', '<base href="./">') | Set-Content build/web/index.html -NoNewline
Copy-Item -Path build/web/* -Destination desktop-shell/web -Recurse -Force
cd desktop-shell
pnpm install
pnpm run dist
```

输出位于 `release-windows-0.9.0/`，包含 `Langbai-Manga-Caption-Studio-Setup-0.9.0.exe`、便携版、`latest.yml` 和差分更新元数据。

## 更新与发布

推送 `v*` 标签会触发 `.github/workflows/release.yml`：在 Windows 上运行分析与测试，构建 Flutter Web、Setup 与便携版，并将安装包和 `latest.yml` 发布到 GitHub Releases。安装版通过 `electron-updater` 下载并安装新版；便携版和其他平台跳转到 Releases 手动更新。

## 设计与技术方向

产品交互参考 Excalidraw 的本地优先工程思路、tldraw 的画布操作、manga-editor-desu 的漫画气泡流程与 Koharu 的文字排版方向；未复制这些项目的界面或源码。
